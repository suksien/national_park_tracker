## A tracker app to record visits and notes to US national parks
* Inspired by the [US national park passport](https://www.nps.gov/olym/planyourvisit/national-parks-passport-program.htm).
* This application is built in Ruby using [Sinatra](https://sinatrarb.com) and communicates to the client via [PUMA](https://puma.io).
* All data are stored in a backend DB using PostgreSQL. User credentials information are first encrypted before being stored in the DB.
